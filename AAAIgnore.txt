# Các thứ mà ta k có thời gian học hay dùng nó nhưng rất hay:

-> Winform:
Winform là free opensource class lib là 1 phần của Microsoft .NET cung cấp nền tảng viết client app giao diện người dùng cho desktop, laptop và tablet

-> Tạo autobot
Trong các fanpage trên facebook, telegram, discord việc tạo bot là kỹ thuật quan trọng nhưng kp là teamdev làm. VD trong trang web của ta, ta cần tạo 1 conbot trả lời tự động hoặc khi trả lời sẽ tự chuyển đến facebook để hỏi bot cũng là kỹ thuật hay. Có tool free của GG cho phép tạo bot

-> Flutter
Flutter phù hợp với các ứng dụng focus về animation, layout phức tạp. React Native cũng cross-platform nhưng animation làm vất vả hơn, dù hiện tại đã có ThreeJS khiến mảng animation của React trở nên cực kỳ mạnh. 
Flutter back bởi GG dùng JS, còn RN back bởi Facebook dùng Dart. 
Flutter xây UX hiệu năng rất cao, giao tiếp gần như TT với native.

-> Hook và lập trình win32:
Hook là 1 cơ chế dùng để chắn ngang các lời gọi hàm, sự kiện trước khi nó đến một ứng dụng. Từ đó có thể sửa đổi theo ý ta, gia tăng hành vi, thêm chức năng or đổi chức năng của ứng dụng. Lập trình hook có thể tùy chỉnh điều này. Nó là 1 kỹ thuật trong lập trình win32
Win32 là viết tắt cho cụm từ Windows 32-bit, Win32 API cung cấp cho ta một số lượng hàm, thủ tục nhất định để ta dễ dàng tạo ra một chương trình window or window SDK theo ý muốn

-> Autocad:
Phần mêm vẽ kỹ thuật bằng đồ họa vector trong kiến trúc, có thể vẽ cấu trúc 3d tòa nhà hay các thứ

-> WebGL:
Có thể sẽ không bao giờ học vì nó giúp code front end nhưng ở tầng level quá thấp nên rất phức tạp, không đủ thời gian. Nó là thứ giúp tương tác với GPU để render đồ họa cao, bao gồm 3D model và các hiệu ứng cực đẹp. K thiếu series dạy.

-> PHP:
Personal Home Page được đổi tên thành Hypertext Preprocessor là ngôn ngữ lập trình kịch bản(scripting language) dùng phổ biến cho các ứng dụng web chạy trên máy chủ (SSR), tức có thể dùng nó từ phía server để sinh ra mã HTML trên client. 
PHP chiếm đa số trong các website SSR, là 1 trong những ngôn ngữ web fullstack phổ biến nhất. Hiện nay rất phổ biến việc sử dụng mã nguồn mở nhúng vào HTML nhờ dùng <?php?> khi áp dụng tối ưu hóa cho các ứng dụng web mang hiệu quả cao, nhanh tiện ít lỗi. 
VD: WordPress, Youtube, Wikipedia hay Facebook dùng PHP. Các website sinh ra từ WordPress cũng dùng PHP.
Do chạy trên server nên cần 1 server ảo để biên dịch lệnh, có nhiều phần mềm như XAMPP, Vertigo giúp tạo server ảo. VD: Vertigo giúp tạo server ảo biên dịch mã PHP, quản lý MySQL, cấu hình domain ảo ip các thứ.
--> Laravel là 1 framework của PHP làm web phổ biến nhất theo mẫu MVC, mã nguồn mở, free. Code được giảm thiểu nhưng vẫn đạt tiêu chuẩn giúp tiết kiệm tg. Nói đến PHP thg nói đến kèm với Laravel.
